## 接口列表

### 1. 生成图片

```
POST /api/generate
Content-Type: application/json
```
**注意事项**
图片生成后保存在本地以供使用，严禁在网页中动态生成加载
**并发**
支持并发生成5张图片，为提高效率，尽量并发生成图片

**请求体：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `prompt` | string | 是 | 图片描述提示词，建议英文效果更佳 |

```json
{
  "prompt": "a red apple on a wooden table"
}
```

**响应：`text/event-stream`（Server-Sent Events）**

推送实时进度到客户端，最终在 `done` 事件中返回图片 URL 或错误。

| 事件 | 数据 | 说明 |
|---|---|---|
| `started` | `{ wmTaskId, providerTaskId, selectedProvider }` | 任务已创建 |
| `poll` | `{ attempt, status }` | 每次轮询结果（`processing` / `succeeded`） |
| `done` | 见下方 | 流结束，带有最终结果或错误 |

`done` 事件成功时的 `data`：

```json
{
  "status": "succeeded",
  "resultUrl": "https://pub-911e4fa03f0c4323a80d8f3dc99d1c7f.r2.dev/effects/16/images/xxxx-watermarked.webp",
  "raw": { /* 原始响应 */ }
}
```

`done` 事件失败时的 `data`：

```json
{
  "status": "failed",
  "error": "generation failed",
  "raw": { /* ... */ }
}
```

可能的 `status` 值：`succeeded`、`failed`、`error`、`timeout`。

**示例（curl）：**

```bash
curl -N -X POST http://localhost:3000/api/generate \
  -H 'Content-Type: application/json' \
  -d '{"prompt":"a red apple on a wooden table"}'
```

## 前端页面

浏览器直接访问 `http://localhost:3000` 可打开带 UI 的图片生成页面，已集成提示词输入、实时状态、图片展示、下载和复制链接功能。

## 调用示例

### JavaScript（浏览器 / Node.js，原生 EventSource）

```js
// 浏览器端直接用 EventSource 无法发 POST，所以用 fetch 流式读取
const resp = await fetch('/api/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ prompt: 'a cute cat wearing a space suit' }),
});
const reader = resp.body.getReader();
const decoder = new TextDecoder();
let buf = '';
while (true) {
  const { value, done } = await reader.read();
  if (done) break;
  buf += decoder.decode(value, { stream: true });
  let sep;
  while ((sep = buf.indexOf('\n\n')) !== -1) {
    const raw = buf.slice(0, sep);
    buf = buf.slice(sep + 2);
    let event = 'message', data = '';
    for (const line of raw.split('\n')) {
      if (line.startsWith('event:')) event = line.slice(6).trim();
      else if (line.startsWith('data:')) data += line.slice(5).trim();
    }
    if (event === 'done') {
      const payload = JSON.parse(data);
      if (payload.status === 'succeeded') {
        console.log('图片地址:', payload.resultUrl);
      } else {
        console.error('失败:', payload.error);
      }
    }
  }
}
```

### Python

```python
import json, sseclient, requests

resp = requests.post(
    "http://localhost:3000/api/generate",
    json={"prompt": "a red apple"},
    stream=True,
)
client = sseclient.SSEClient(resp)
for event in client.events():
    data = json.loads(event.data)
    if event.event == "done":
        print(data)           # {'status':'succeeded','resultUrl':'...'}
        break
```

需要先安装 `sseclient-py`：`pip install sseclient-py requests`。

## 限制与注意事项

- 生成耗时通常在 **15–60 秒**，轮询间隔 5 秒，超时上限 5 分钟。
- 上游服务可能在高峰时段响应变慢或失败，可多次重试
